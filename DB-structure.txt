=========== Teams compositions ===============

--------------------------------------

=================== lineups ==================

G2008_lineups
-------------
ID 					Unique ID
teams_ID			Related team's ID
Type 				7er, 9er (?)
Schema				4_3_1, 3_3_2

G2008_lineup_details
---------------------
ID 					Unique ID
lineup_ID			Related lineup's ID
player_ID			Player's ID
Role 				GOAL_KEEPER, CENTRE_BACK_LEFT, ... SUBSTITUTE
